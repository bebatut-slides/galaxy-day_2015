# ASaiM framework Bioinformatics framework to generate workflows to analyze data from gut microbiota
### Tests of conception * Simple Python scripts * Worflow managers such as Luigi, Airflow, ... * Homemade approach * Configuration file * Workflow description * Web interface for generation * Python scripts to execute workflow * Galaxy instance
### To launch the instance * Get the code source from [Github](https://github.com/ASaiM/framework) ``` $ git clone git@github.com:ASaiM/framework.git ``` * Install the required tools found in README file * Launch the interface ``` $ cd framework $ ./scripts/launch_galaxy.sh ``` * Visualize it: [http://127.0.0.1:8080/](http://127.0.0.1:8080/)
